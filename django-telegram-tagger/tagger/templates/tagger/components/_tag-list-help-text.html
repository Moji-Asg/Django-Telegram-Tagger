Setting tag list text using special keywords.


Keywords list:

{MENTION()} User mention tags by text inside parentheses
{FIRSTNAME} User's first name
{LASTNAME} User's last name (will be empty if user had no last name)
{USERNAME} User's username (will be empty if user had no username)
{USERID} User's ID
{GAMEURL()} Join game link (only if auto tag is enabled)


Decoration keywords

** ** For bold text (Example: **Hello**   ←  Hello)
__ __ For italic text (Example: __Hello__   ←  Hello)
-- -- For underline (Example: --Hello--  ←  Hello)
~~ ~~ For strikethrough (Example: ~~Hello~~  ←  Hello)
` ` For sending text as code
``` ``` For sending text as code (multi line codes)
|| || For sending spoiler texts
